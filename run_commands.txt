编译 source环境变量
cd /workspace/ros2_ws
colcon build && source install/setup.bash

--------------------------------------
编译指定 && --symlink-install 创建符号链接，这样修改Python脚本后无需重新编译即可生效
colcon build --packages-select <指定python包名，可多个（应该）> --symlink-install
e.g.
colcon build --packages-select param_demo --symlink-install && source install/setup.bash
--------------------------------------

每个功能包单独运行命令

1. hello_world_demo
ros2 run hello_world_demo hello_world

2. topic_demo
ros2 run topic_demo topic_pub
ros2 run topic_demo topic_sub

3. service_demo
ros2 run service_demo service_server
ros2 run service_demo service_client 5 10

4. action_demo
ros2 run action_demo action_server
ros2 run action_demo action_client

5. param_demo
ros2 run param_demo param_demo_node
ros2 param set param_demo_node my_param changed_value05102051

6. DDS_qos_demo
单独编译 colcon build --packages-select DDS_qos_demo --symlink-install && source install/setup.bash
ros2 run DDS_qos_demo dds_qos_pub_node
ros2 run DDS_qos_demo dds_qos_sub_node

7 launch_demo  # 运行多个节点
单独编译 
colcon build --packages-select launch_demo --symlink-install && source install/setup.bash
ros2 launch launch_demo single_node.launch.py
ros2 launch launch_demo remapping.launch.py
ros2 launch launch_demo param.launch.py
ros2 launch launch_demo param_yaml.launch.py
